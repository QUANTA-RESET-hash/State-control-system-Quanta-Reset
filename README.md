# State-control-system-Quanta-Reset
System for rapid state switching through physical actions

Краткое описание:
Данная система представляет собой набор телесных действий, направленных на быстрое изменение психоэмоционального состояния человека в реальном времени.
Система применяется в условиях стресса, тревоги, перегрузки и потери контроля.
________________________________________
Принцип работы:
Жест → Дыхание → Внимание → Изменение состояния
________________________________________
Структура системы:
•	Базовая архитектура (описана в system-core.pdf) 
•	Набор драйверов (описан в drivers-catalog.pdf) 
•	Каждый драйвер содержит несколько режимов применения 
________________________________________
Назначение:
Система предназначена для быстрого восстановления контроля над состоянием без длительных практик и теоретической подготовки.
